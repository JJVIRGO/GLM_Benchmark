# Experiment 3 — Data Sources

All ChIP-seq peak files were downloaded from [CistromeDB](http://cistrome.org/db/) as pre-processed BED files. DNase BigWig signal files were downloaded from [ENCODE](https://www.encodeproject.org/). The accession IDs listed below allow tracing back to the original deposited data in NCBI GEO or ENCODE.

> **Note**: CistromeDB provides processed BED files derived from GEO submissions. The GEO accessions listed here are the upstream source experiments that CistromeDB references.
